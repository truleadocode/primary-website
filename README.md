# Truleado Platform

A Next.js platform connecting businesses with influencers, featuring AI-powered search, campaign management, and comprehensive analytics.

## Overview

Truleado is a full-stack application built with Next.js 14+ and Supabase that enables businesses to find and collaborate with influencers through an intuitive platform with AI-powered search capabilities.

## Key Features

- **Dual Search System**: Simple database search and AI-powered conversational search
- **Campaign Management**: Request-based campaign system with contract negotiation
- **Project Organization**: Projects with shortlisted influencers and saved search history
- **Team Collaboration**: Role-based access control (Owner, Admin, Teammate)
- **Analytics Dashboards**: Spend tracking for businesses, earnings tracking for influencers
- **Subscription Billing**: Paddle integration with 7-day free trial and $49/month per seat
- **Real-time Messaging**: Business-influencer communication
- **Comprehensive Admin System**: Backend management for all platform features

## Tech Stack

- **Frontend**: Next.js 15+ (App Router) with TypeScript and React 19
- **Backend**: Supabase (PostgreSQL, Authentication, Real-time)
- **AI Search**: OpenRouter API
- **Billing**: Paddle
- **Email**: Resend
- **Styling**: Tailwind CSS
